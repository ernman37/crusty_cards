# Security Policy

## Reporting a Vulnerability

If you find a security issue, please open a [GitHub issue](https://github.com/ernman37/rusty_cards/issues) or email the maintainer directly.

## Security Note

This library uses the `rand` crate for shuffling. While suitable for games and simulations, it is **not cryptographically secure**.

**Do not use for:**
- Real-money gambling
- Cryptographic purposes
- Security-sensitive applications
